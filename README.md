# IPO Valuation Predictor & Market Readiness Dashboard

## Project Overview
This project is an end-to-end predictive analytics pipeline designed to evaluate a company's financial readiness to go public. By isolating intrinsic financial value from market-driven hype, this tool helps identify key pricing variances across multiple industry sectors.

## Tech Stack
* **Python (Scikit-Learn, Pandas):** Engineered a calibrated Linear Regression model to forecast base market pricing based on core financial fundamentals (Revenue Growth, EBITDA).
* **Power BI:** Developed an interactive business intelligence dashboard to visualize data and benchmark sector viability.
* **Microsoft Excel:** Served as the dynamic database for storing historical and current target company metrics.

## Key Features & Impact
* **Automated Pipeline:** Processes 500+ financial data points across 100 historical IPOs spanning 10 distinct sectors.
* **Predictive Accuracy:** The machine learning model achieved an 82% predictive accuracy ($R^2$ score) for base valuations.
* **Dynamic Visualization:** Features live, cross-filtering parameters to evaluate market entry conditions in real-time (e.g., contrasting the Financials sector at a 62.5% readiness score against Health Care at 12.5%).
